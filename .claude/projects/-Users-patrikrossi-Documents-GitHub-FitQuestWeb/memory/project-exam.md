FitQuestWeb e il progetto d'esame per Applicazioni Web, Mobile e Cloud (Modulo 2) a UNICAM.

Requisiti esame:
- App web cloud-native completa (frontend + backend + database)
- Repository pubblico GitHub con: README completo, diagrammi architettura/deploy, file YAML config, pipeline CI/CD, link deployment o istruzioni local
- Consegna almeno 7 giorni prima della data esame a christian.lillini@unicam.it
- Presentazione finale: pitch 10 minuti (slide + demo live consigliata)
- Il README deve essere chiaro e completo per build from scratch

**Why:** Le priorita di review vanno calibrate su cosa valuta il professore (completezza, architettura pulita, documentazione, CI/CD funzionante, demo live) non su standard production.

**How to apply:** Dare peso a: qualita codice dimostrabile, test che mostrano padronanza, documentazione, diagrammi, CI/CD pipeline, UX della demo. Dare meno peso a: hardened security, performance optimization, scalabilita.
